# Kovyl

**Kovyl** /ˈkɒvɪl/ — это статически типизированный мультипарадигмальный язык программирования с ручным управлением памятью и дружелюбным синтаксисом.

Язык из коробки предлагает полезные операции над данными в виде **спешлей** (*specials*) и не содержит ни одной встроенной функции. Это делает код на языке явным и читаемым.

Несмотря на скромный вид синтаксиса, в Kovyl присутствует минималистичный сахар и ряд мощных решений, делающих проектирование и скриптинг не только мощным, но и приятным.

---

## Область применения

Язык полезен там, где нужен простой, ясный код с готовыми решениями. Обширная стандартная библиотека и бэкенд языка способны предоставить всё необходимое.

---

## Возможности

### Функциональное программирование

В Kovyl функции — объекты первого класса. Это позволяет передавать их как значения, присваивать переменным и вызывать из любого хранилища.

### Самодостаточность

Встроенные спешлы дают всё то, что было бы без них невозможно: ввод/вывод, управление памятью, форматирование. Огромный выбор утилит из стандартной библиотеки позволяет писать, не задумываясь о простых алгоритмах. При этом библиотеки чаще всего написаны на самом Kovyl.

### Семантика

Нисходящий анализ типов, неявное расширение размера массива, явные конвертации между типами, строки с нуль-терминатором и типизированные паники делают код безопасным и удобным.

### Инструментарий

LSP-сервер с семантическим и синтаксическим анализаторами позволяет писать код быстрее, не запуская его каждый раз. Читаемое форматирование ошибок не даёт задуматься о причине надолго.

Режим отладки компилятора (флаг `-d`) позволяет ясно отследить, как изменялись и обрабатывались синтаксические конструкции. Краткие ошибки (флаг `-s`) созданы для быстрого поиска.

## Быстрый старт

Всё, что нужно для запуска: **Nim** и скопированный репозиторий.

```bash
nimble run kovyl -- your_script.kvl
```

Простой пример:

```kovyl
func string greeting(string name) do 
  return fmt:("Hello from Kovyl, ", name, "!")
end

string@ names = {v"Alice", v"Ben", v"John"}

for name = names do
  print:(greeting(name))
end
```

## Философия

Kovyl предлагает тебе диктовать правила. Ты не борешься с компилятором — ты создаёшь код, который компилятор слушает. Ты управляешь рантаймом. Никаких скрытых аллокаций, отлавливаемые паники, мощнейшая система типов — всё это в комплекте даёт не просто инструмент, а целую мастерскую возможностей.
